# Pit-Wall

F1 팬용 일정·순위 웹앱 (MVP: F-01~F-05)

- 시즌 전체 일정을 한국시간(KST)으로 변환해 표시
- 다음 세션까지 실시간 카운트다운
- 드라이버 / 컨스트럭터 포인트 순위
- 세션 시작 1시간·10분 전 브라우저 알림

데이터 출처: [Jolpica-F1 API](https://github.com/jolpica/jolpica-f1) (Ergast 호환, 무료·무인증)

## 배포

GitHub Pages로 배포: Settings → Pages → Source를 `main` / `(root)`로 설정하면
`index.html`이 자동으로 루트에서 서빙됩니다.
